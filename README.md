# RNA-seq-Analysis
## 0. touch a new `sample.txt`
~~~
$ touch sample.txt
- sample1
- sample2
- sample3
- sample4
- sample5
~~~
## 1. mkdir `fastq` folder
mkdir a new `fastq` folder to store your all sequence files
~~~
$ mkdir fastq
$ mv *.fastq.gz fastq/
~~~
## 2. run shell scripts 
~~~
$ bash ./script/RNA-seq_analysis
~~~
